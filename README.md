Personal Voice Assistant (Python)

A simple yet functional voice-controlled assistant built using Python that can understand user commands and perform basic automation tasks.

##Features

* 🎙️ Speech-to-text command recognition
* 🔊 Text-to-speech responses
* 🌐 Open websites (e.g., Chrome/YouTube)
* 📸 Take screenshots
* 📝 Task management (add & list tasks)
* 💬 Basic conversational responses

##Tech Stack

* Python
* SpeechRecognition
* gTTS (Google Text-to-Speech)
* PyAutoGUI
* Pillow (for screenshots)

##How It Works

1. Listens to user voice input
2. Converts speech to text
3. Matches commands using rule-based logic
4. Executes the corresponding action
5. Responds using voice output

##Getting Started

### Install dependencies:

bash
pip install SpeechRecognition gTTS pyautogui pillow playsound


### Run the project:

bash
python assistant.py
